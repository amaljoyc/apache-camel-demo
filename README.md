# apache-camel-demo
SpringBoot app in Kotlin to showcase Camel sample integrations

Camel Components used in this project are,
1. camel-rabbitmq
2. camel-mongodb
3. camel-csv
4. camel-file

To run the app, execute the command
```
./gradlew bootrun
```

Details blog post - https://amaljoyc.com/blog/2019/apache-camel/
